Juno is a powerful, free environment for the Julia language.

Read more about it below or get going straight away.

Installation »

Walks like Python. Runs like C.

Juno fully exploits Julia's unique combination of ease-of-use and performance. Immediate, interactive feedback lets non-experts to build reliable programs and get results quickly.

Useful, real-time feedback.

With a completely live environment, Juno aims to take the frustration and guesswork out of programming and put the fun back in.

Great for power users, too.

Juno is built on Light Table, which means it's a decent editor as well as a pretty face. It's customisable to your heart's content and has plenty of power features; multiple cursors, fuzzy file finding, Emacs keybindings, you name it.
